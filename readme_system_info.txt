NAME
	system_info - System Information
SYNOPSIS
	system_info [argument]
DESCRIPTION
	system_info calls system utilities based on the argument entered. 
	It uses if else statements to assess the argument and match the to
	the relevant code block.  Only one argument can be processed with
	each call to system_info. 
OPTIONS
	*Enter as arguments without any flag*
	memory
		Displays the amount of free and occupied memory. 
	disk
		Amount of free and occupied disk space. 
	connection
		Connection information to the system.
	time
		The amount of time the system has been running.
